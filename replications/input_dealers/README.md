### Abstract

Access to fertilizer, seed, and agrochemical retailers is a binding constraint on technology adoption for many smallholder farmers in Ghana. This study links the geocoded agro-dealer sale points dataset (`ghana_agro_dealer_sale_points.rds`) to harmonized GLSS farmer data and uses a threshold-search routine plus a meta-stochastic frontier framework to test whether proximity to input dealers raises farm-level technology adoption and technical efficiency. *Full abstract to follow once v000 lands.*
