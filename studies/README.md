## Overview

For an overview of the broader project context, please refer to the main
[okwaayeli README](https://github.com/ftsiboe/okwaayeli/blob/main/README.md)
in the repository root.

See the
[LICENSE](https://github.com/ftsiboe/okwaayeli/blob/main/LICENSE)
file in the repository root for details.

This directory holds one folder per study. Each study folder contains the
analysis pipeline (data assembly, matching, meta-stochastic frontier
estimation, and exhibits) together with the study's manuscript and supporting
material, so that a study's code and its write-up live in one place. The
objective is a transparent and reproducible framework that supports, validates,
and extends the findings reported in the associated publications.

Note that the [legacy
studies](https://github.com/ftsiboe/okwaayeli/tree/main/studies/legacy_studies)
were conducted before the project fully transitioned to GitHub. As such,
their content is not based on the current resources available in this
repository. Nonetheless, legacy files are provided in the individual
study folders to reproduce, as closely as possible, the initial studies.

## Structure

Each folder contains the scripts, data, and results specific to the study
indicated by its name. The individual studies include:

- **`ag_services`**: Evaluates the effect of agricultural services on farm
  production.
- **`disability`**: Investigates the impact of disabilities on farm
  production.
- **`education`**: Analyzes how education influences farm production.
- **`financial_inclusion`**: Examines how financial inclusion affects farm
  production.
- **`income_transfer`**: Explores how income transfers affect farm
  production.
- **`input_dealers`**: Investigates the role of agricultural input dealers.
- **`land_tenure`**: Assesses how land tenure security influences farm
  production.
- **`resource_extraction`**: Investigates the impact of resource extraction
  activities on farm production.
- **`societal_peace_and_cohesion`**: Examines how conflict and social
  cohesion affect farm production.
- **`time_poverty`**: Examines how time poverty relates to farm production.
- **`legacy_studies`**: Pre-GitHub studies, retained for reference.
